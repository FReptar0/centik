**Props interface:**
```typescript
interface FloatingInputProps {
  label: string
  value: string
  onChange: (value: string) => void
  type?: 'text' | 'number' | 'decimal' | 'date'
  prefix?: string       // "$" for money inputs, static, does not float
  suffix?: string       // "%" for rate inputs, static
  error?: string        // shows below input in Meta/negative style
  optional?: boolean    // shows "(opcional)" in label
  id?: string           // custom id, auto-generated if not provided
  name?: string
  className?: string
  disabled?: boolean
  autoFocus?: boolean
}
```

**Implementation details:**
- "use client" directive (manages focus state)
- Wrapper: `<div className="relative">` containing label, optional prefix/suffix, input, and error text
- Generate unique id via `useId()` from React if not provided via props
- Track focus state with `useState<boolean>(false)` via onFocus/onBlur handlers
- Determine `isFloating` state: `isFocused || value.length > 0`

**Label positioning (CSS approach using Tailwind classes):**
- Label is `<label>` with `htmlFor={id}` positioned absolute inside the relative wrapper
- Default position (not floating): `top-[10px]` (at input text baseline), `text-sm` (14px), `text-text-tertiary`, `pointer-events-none`
- Floating position: `top-[-8px]` (above input), `text-[11px]`, `uppercase`, `tracking-[2px]`, `text-text-secondary`
- Transition: `transition-all duration-200 ease-out`
- When optional: append " (opcional)" to label text in floating state only

**Input styling:**
- `bg-transparent` (no background)
- `border-0 border-b` (bottom border only)
- Default border: `border-border-divider`
- Focused border: `border-accent`
- Error border: `border-negative`
- `text-text-primary text-sm` (14px Body)
- `py-[10px] px-0` (padding per spec: 10px vertical, 0 horizontal)
- `w-full outline-none` (remove default focus outline, handled by underline color)
- `font-sans` (inherits Satoshi from body)
- If prefix: add `pl-5` (20px left padding to make room for prefix)
- If suffix: add `pr-5` (20px right padding)

**Prefix/Suffix:**
- Prefix: `<span>` with `absolute left-0 top-[10px] text-text-tertiary text-xs pointer-events-none`
- Suffix: `<span>` with `absolute right-0 top-[10px] text-text-tertiary text-xs pointer-events-none`

**Error display:**
- Below input: `<p className="mt-1 text-[11px] text-negative">{error}</p>`
- Only rendered when error prop is truthy

**onChange handler:** Call `onChange(e.target.value)` -- the component is fully controlled.

**Important:** Do NOT use the global :focus-visible outline for this component. The input's focus is communicated via the underline color change, not an outline. Override with `focus-visible:outline-none` on the input element if needed.
