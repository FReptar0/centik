       ```css

       --animate-status-pulse: status-pulse 2.5s ease-in-out infinite;
       @keyframes status-pulse {
         0%, 100% { opacity: 1; transform: scale(1); }
         50% { opacity: 0.5; transform: scale(0.85); }
       }

       /* Animation: pixel-dissolve scanline reveal */
       --animate-scanline-reveal: scanline-reveal 500ms steps(12, end) forwards;
       @keyframes scanline-reveal {
         0% { clip-path: inset(0 0 100% 0); }
         100% { clip-path: inset(0 0 0% 0); }
       }
       ```
       CRITICAL: The `--animate-*` token name and the keyframe reference MUST be on the same line (Tailwind v4 requirement per PITFALLS.md).

       ```css

         --font-sans: var(--font-satoshi);
       ```

    5. **Global styles OUTSIDE @theme**:
