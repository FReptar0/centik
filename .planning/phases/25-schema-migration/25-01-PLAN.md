**Step C: Update seed.ts** with admin user + userId backfill:

2. Add `seedAdminUser()` function BEFORE all other seed functions:

   - Read password from `process.env.ADMIN_PASSWORD` or default to `'centik-dev-2026'` (dev only)

3. Pass `adminUserId` to every seed function. Each function must include `userId: adminUserId` in every `create` call:

   - `seedPeriods(adminUserId)` -- add userId to both period upserts

   - `seedDebts(adminUserId)` -- add userId to every debt upsert
   - `seedValueUnits(adminUserId)` -- add userId to every value unit upsert
   - `seedBudgets(categories, currentPeriodId, adminUserId)` -- add userId to every budget upsert
   - `seedTransactions(categories, currentPeriodId, previousPeriodId, incomeSources, adminUserId)` -- pass userId to buildCurrentMonthTransactions and buildPreviousMonthTransactions
   - `seedMonthlySummary(previousPeriodId, adminUserId)` -- add userId to create
   - `seedUnitRates(valueUnits, adminUserId)` -- add userId to every unit rate create
