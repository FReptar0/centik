Prisma Debt model fields:
```prisma
model Debt {
  id              String   @id @default(cuid())
  name            String   @unique
  type            DebtType
  currentBalance  BigInt
  creditLimit     BigInt?
  annualRate      Int      // basis points (4500 = 45.00%)
  minimumPayment  BigInt?
  monthlyPayment  BigInt?
  originalAmount  BigInt?
  remainingMonths Int?
  cutOffDay       Int?
  paymentDueDay   Int?
  isActive        Boolean  @default(true)
  createdAt       DateTime @default(now())
  updatedAt       DateTime @updatedAt
}
```

DFR.md calculated fields:
- utilizationRate: currentBalance / creditLimit (credit cards only)
- estimatedMonthlyInterest: currentBalance * (annualRate / 12 / 10000)
- percentPaid: 1 - (currentBalance / originalAmount) (loans only)
- totalRemainingPayment: monthlyPayment * remainingMonths (loans only)
