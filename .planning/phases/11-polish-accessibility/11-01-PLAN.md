  <action>
Add `import { toast } from 'sonner'` to each file, then add toast calls at every mutation result point. All toast messages in Spanish. Pattern:

**For components using ActionResult pattern** (IncomeSourceForm, CategoryForm, TransactionForm, DebtForm, PresupuestoClientWrapper):

After `if ('success' in result)`:
```ts
toast.success('Mensaje de exito')
```

In the `else` (error) branch:
```ts
const messages = Object.values(result.error).flat()
toast.error(messages[0] ?? 'Error al guardar')
```

**For delete actions in cards** (IncomeSourceCard, CategoryList, TransactionRow, DebtCard):
Delete actions currently do `await deleteX(id)` without checking result. Wrap in try/catch:
```ts
try {
  const result = await deleteX(id)
  if (result && 'error' in result) {
    const messages = Object.values(result.error).flat()
    toast.error(messages[0] ?? 'Error al eliminar')
  } else {
    toast.success('Mensaje eliminado')
  }
} catch {
  toast.error('Error al eliminar')
}
```

**For reopen/close period** (HistorialClientWrapper, PresupuestoClientWrapper, MovimientosClientWrapper):
Currently call `await reopenPeriod(id)` or `await closePeriod(id)` without checking result. Add result check:
```ts
const result = await reopenPeriod(periodId)
if ('success' in result) {
  toast.success('Periodo reabierto')
  router.refresh()
} else {
  const messages = Object.values(result.error).flat()
  toast.error(messages[0] ?? 'Error al reabrir periodo')
}
```
