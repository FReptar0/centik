   - Return true on success

5. `upsertBudgets(periodId: string, data: unknown): Promise<ActionResult>`
   - Validate data with createBudgetSchema.safeParse
   - On validation failure, return { error: parsed.error.flatten().fieldErrors }
   - For each entry in parsed.data.entries, use prisma.budget.upsert with:
     - where: { periodId_categoryId: { periodId, categoryId: entry.categoryId } }
     - update: { quincenalAmount: BigInt(entry.quincenalAmount) }
     - create: { periodId, categoryId: entry.categoryId, quincenalAmount: BigInt(entry.quincenalAmount) }
   - Use Promise.all to run upserts in parallel (each upsert targets a different unique composite)
   - On success, call revalidateBudgetPaths() and return { success: true }
   - Catch errors, return { error: { _form: ['Error de servidor'] } }
