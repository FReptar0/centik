    Create `src/lib/income.ts` with two exported functions:

    1. `getMonthlyEquivalent(defaultAmount: string, frequency: string): string`
       - Converts a source's amount to its monthly equivalent using BigInt arithmetic
       - QUINCENAL: amount * 2n
       - SEMANAL: amount * 4n (approximate: ~4 weeks/month)
       - MENSUAL: amount (no conversion)
       - VARIABLE: amount (use defaultAmount as estimate; INC-05 says show "(estimado)" in UI since transactions don't exist yet)
       - Returns the result as string (serialized centavos)

    2. `calculateIncomeSummary(sources: SerializedIncomeSource[]): IncomeSummary`
       - Takes serialized income sources (defaultAmount is string)
       - Accumulates quincenal total using BigInt:
         - QUINCENAL sources: add defaultAmount directly
         - SEMANAL sources: add defaultAmount * 2n (2 weeks per quincena)
         - MENSUAL sources: add defaultAmount / 2n (half per quincena)
         - VARIABLE sources: add defaultAmount / 2n (estimate, half per quincena)
       - Returns { quincenal, monthly, semester, annual } as strings where:
         - monthly = quincenal * 2n
         - semester = quincenal * 12n
         - annual = quincenal * 24n

    Implement three Server Actions:

    1. `createIncomeSource(data: unknown): Promise<ActionResult>`
       - Parse with `createIncomeSourceSchema.safeParse(data)`
       - On validation failure: return `{ error: parsed.error.flatten().fieldErrors }`
       - On success: `prisma.incomeSource.create({ data: { name, defaultAmount: BigInt(parsed.data.defaultAmount), frequency, type } })`
       - Catch Prisma P2002 unique constraint: return `{ error: { name: ['Ya existe una fuente con ese nombre'] } }`
       - Call `revalidatePath('/ingresos')` and `revalidatePath('/')`
       - Return `{ success: true }`

       - Parse with `createIncomeSourceSchema.safeParse(data)` (same schema for update)

       - Catch Prisma P2025 (record not found): return `{ error: { _form: ['Fuente de ingreso no encontrada'] } }`
       - Catch Prisma P2002 (duplicate name): return friendly error
       - Call `revalidatePath('/ingresos')` and `revalidatePath('/')`
       - Return `{ success: true }`

    3. `deleteIncomeSource(id: string): Promise<ActionResult>`
       - `prisma.incomeSource.delete({ where: { id } })`
       - Catch Prisma P2025: return `{ error: { _form: ['Fuente de ingreso no encontrada'] } }`
