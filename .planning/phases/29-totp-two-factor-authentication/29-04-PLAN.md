<objective>
Deliver the user-facing 2FA lifecycle UI — setup, disable, regenerate — plus the 4 Server Actions that back them. All writes that touch >1 row use `prisma.$transaction` per CLAUDE.md security rules and Phase 28 P03 hash-before-transaction learning.

**Scope:**
1. **4 Server Actions** in `src/app/(app)/configuracion/totp-actions.ts`:
   - `prepareTotpSecretAction()` — generate TOTP secret + QR data URL (no DB writes; client holds secret in memory until step 2)
   - `enableTotpAction(formData)` — verify code against proposed secret, hash 10 backup codes BEFORE opening $transaction, atomically persist encrypted secret + set `totpEnabled=true` + create 10 BackupCode rows
   - `disableTotpAction(formData)` — verify current code (TOTP or backup), atomically clear `totpSecret` + `totpEnabled=false` + deleteMany BackupCode
   - `regenerateBackupCodesAction(formData)` — verify current code, atomically deleteMany + createMany new 10 rows

2. **1 Seguridad section** in ConfiguracionClientWrapper (visible to every authenticated user per D-19, NOT admin-gated).

3. **3 bottom-sheet modals**:
   - `Activar2faModal` — 3-step wizard (Escanear → Verificar → Guardar codigos)
   - `Desactivar2faModal` — single-step, destructive confirm
   - `RegenerarCodigosModal` — single-step + codes display

4. **1 shared `BackupCodesScreen`** — 10 codes as `XXXX-XXXX` monospace grid, Copiar button, Descargar (.txt) button, mandatory checkbox gating "Listo".

5. **1 page/wrapper edit** — `page.tsx` fetches `totpEnabled`, wrapper renders `<SeguridadSection />`.

**Purpose:** This is the only path for a user to enable/disable 2FA. Without it, TOTP-01, TOTP-02 partial (the at-rest encryption happens here via enableTotpAction), and TOTP-04 (backup codes) are not delivered.

**Output:** ~600 lines of production TSX + 4 Server Actions + paired unit tests bringing all 7 Wave-0 totp-actions stubs to green.
</objective>

    ```typescript
    'use server'

    import { revalidatePath } from 'next/cache'
    import QRCode from 'qrcode'
    import bcrypt from 'bcryptjs'
    import prisma from '@/lib/prisma'
    import { requireAuth } from '@/lib/auth-utils'
    import { enableTotpSchema, disableTotpSchema } from '@/lib/validators'
    import { createTotpSecret, buildOtpauthUri, verifyTotp } from '@/lib/totp'
    import { encryptSecret, decryptSecret } from '@/lib/totp-crypto'
    import {
      generateBackupCodes,
      formatForDisplay,
      consumeBackupCode,
    } from '@/lib/backup-codes'

    type PrepareResult =
      | { secret: string; qrDataUrl: string }
      | { error: string }

    type EnableResult =
      | { success: true; backupCodes: string[] }
      | { error: Record<string, string[]> }

    type SimpleFormResult =
      | { success: true }
      | { error: Record<string, string[]> }

    type RegenResult =
      | { success: true; backupCodes: string[] }
      | { error: Record<string, string[]> }

    /**
     * Step 1 of 2FA setup (Phase 29 D-20 step 1).
     * Generates a fresh TOTP secret + QR data URL. NO DB writes.
     * Secret is held in client memory until step 2 submits it back with the first verification code.
     */
    export async function prepareTotpSecretAction(): Promise<PrepareResult> {
      const { userId } = await requireAuth()
      try {
        const user = await prisma.user.findUnique({
          where: { id: userId },
          select: { email: true },
        })
        if (!user) return { error: 'No autorizado' }

        const secret = createTotpSecret()
        const uri = buildOtpauthUri(secret, user.email)
        const qrDataUrl = await QRCode.toDataURL(uri, {
          errorCorrectionLevel: 'M',
          margin: 2,
          width: 240,
        })
        return { secret, qrDataUrl }
      } catch {
        return { error: 'No pudimos preparar 2FA. Intenta de nuevo.' }
      }
    }

    /**
     * Step 2 of 2FA setup (Phase 29 D-20 step 2, Pattern 9).
     * Verifies the code against the proposed secret, hashes 10 backup codes, and
     * atomically: (a) encrypts+persists secret, (b) flips totpEnabled, (c) creates BackupCode rows.
     *
     * hash BEFORE $transaction — bcrypt cost 12 × 10 = ~3s of DB connection pinning if inside (Pitfall 9).
     * Returns plaintext backup codes ONCE — client step 3 displays them.
     */
    export async function enableTotpAction(
      _prev: EnableResult | undefined,
      formData: FormData,
    ): Promise<EnableResult> {
      const { userId } = await requireAuth()

      const parsed = enableTotpSchema.safeParse({
        secret: formData.get('secret'),
        code: formData.get('code'),
      })
      if (!parsed.success) return { error: parsed.error.flatten().fieldErrors }

      // Verify code against the PROPOSED secret (not yet stored)
      if (!(await verifyTotp(parsed.data.secret, parsed.data.code))) {
        return { error: { code: ['Codigo invalido'] } }
      }

      // Hash BEFORE opening the transaction (Phase 28 P03 rule)
      const plainCodes = generateBackupCodes(10)
      const hashes = await Promise.all(
        plainCodes.map((c) => bcrypt.hash(c.toLowerCase(), 12)),
      )
      const encryptedSecret = encryptSecret(parsed.data.secret)

      try {
        await prisma.$transaction([
          prisma.user.update({
            where: { id: userId },
            data: { totpSecret: encryptedSecret, totpEnabled: true },
          }),
          prisma.backupCode.deleteMany({ where: { userId } }), // defensive: clear stale
          prisma.backupCode.createMany({
            data: hashes.map((codeHash) => ({ userId, codeHash })),
          }),
        ])
      } catch {
        return { error: { _form: ['No pudimos activar 2FA. Intenta de nuevo.'] } }
      }

      revalidatePath('/configuracion')
      return {
        success: true,
        backupCodes: plainCodes.map(formatForDisplay),
      }
    }

    /**
     * Disable 2FA (Phase 29 D-21). Requires a current TOTP code OR backup code.
     * Atomically clears totpSecret, flips totpEnabled=false, and deletes all BackupCode rows.
     * Private helper verifyCurrentCode returns true iff the code is a valid TOTP or consumable backup.
     */
    export async function disableTotpAction(
      _prev: SimpleFormResult | undefined,
      formData: FormData,
    ): Promise<SimpleFormResult> {
      const { userId } = await requireAuth()

      const parsed = disableTotpSchema.safeParse({ code: formData.get('code') })
      if (!parsed.success) return { error: parsed.error.flatten().fieldErrors }

      const ok = await verifyCurrentCode(userId, parsed.data.code)
      if (!ok) return { error: { _form: ['Codigo invalido'] } }

      try {
        await prisma.$transaction([
          prisma.user.update({
            where: { id: userId },
            data: { totpSecret: null, totpEnabled: false },
          }),
          prisma.backupCode.deleteMany({ where: { userId } }),
        ])
      } catch {
        return { error: { _form: ['No pudimos desactivar 2FA. Intenta de nuevo.'] } }
      }

      revalidatePath('/configuracion')
      return { success: true }
    }

    /**
     * Regenerate backup codes (Phase 29 D-13, D-22). Requires a current TOTP or backup code.
     * Atomically deletes existing rows + inserts a fresh 10.
     */
    export async function regenerateBackupCodesAction(
      _prev: RegenResult | undefined,
      formData: FormData,
    ): Promise<RegenResult> {
      const { userId } = await requireAuth()

      const parsed = disableTotpSchema.safeParse({ code: formData.get('code') })
      if (!parsed.success) return { error: parsed.error.flatten().fieldErrors }

      const ok = await verifyCurrentCode(userId, parsed.data.code)
      if (!ok) return { error: { _form: ['Codigo invalido'] } }

      const plainCodes = generateBackupCodes(10)
      const hashes = await Promise.all(
        plainCodes.map((c) => bcrypt.hash(c.toLowerCase(), 12)),
      )

      try {
        await prisma.$transaction([
          prisma.backupCode.deleteMany({ where: { userId } }),
          prisma.backupCode.createMany({
            data: hashes.map((codeHash) => ({ userId, codeHash })),
          }),
        ])
      } catch {
        return { error: { _form: ['No pudimos regenerar los codigos. Intenta de nuevo.'] } }
      }

      revalidatePath('/configuracion')
      return { success: true, backupCodes: plainCodes.map(formatForDisplay) }
    }

    /** Private helper — verifies a code against stored secret OR backup codes. */
    async function verifyCurrentCode(userId: string, code: string): Promise<boolean> {
      const user = await prisma.user.findUnique({
        where: { id: userId },
        select: { totpSecret: true, totpEnabled: true },
      })
      if (!user?.totpEnabled || !user.totpSecret) return false

      if (/^\d{6}$/.test(code)) {
        const secret = decryptSecret(user.totpSecret)
        return verifyTotp(secret, code)
      }
      const normalized = code.replace(/-/g, '').trim().toLowerCase()
      if (/^[0-9a-f]{8}$/.test(normalized)) {
        return consumeBackupCode(userId, normalized)
      }
      return false
    }
    ```
