   CRITICAL decisions:
   - Do NOT use `--color-*: initial` to reset the namespace (per research pitfall #6). This preserves default Tailwind colors like white, black, transparent that Recharts tooltips and third-party components may use.
   - Use `@theme` (NOT `@theme inline`) for all static hex values.
   - Use `@theme inline` ONLY for `--font-sans: var(--font-dm-sans)` because it references a CSS variable that must be resolved at runtime (per research pitfall #2).
   - Remove ALL `:root` variables and `@media (prefers-color-scheme: dark)` blocks. Dark mode is static -- no light mode exists.
