**Charts spec (replaces existing):**

Rename to: "### Charts"

Container: Background `--color-surface-elevated`. Padding 20px. Border-radius `--radius-lg`.

Line/Area charts:
- Stroke width: 1.5px (reduced from 2px for minimal feel).
- Dot endpoints: 4px solid dots on each data point. Same color as the line stroke.
- Area fill: Gradient from stroke color at 10-15% opacity at top to 0% at bottom. Subtle, not dominant.
- No grid lines. Remove all CartesianGrid references.
- Axis labels: Minimal. X-axis shows only start and end date labels for time series. No Y-axis labels. Values shown on hover/tooltip only.
- Axis ticks: `--color-text-tertiary`, 11px, Meta level.
- Axis lines: hidden.

Bar charts:
- Rectangular bars, no border-radius (flat tops). Clean geometric fill.
- Bar width: Thinner than default -- approximately 60% of available space to show gaps between bars.
- No outlines or strokes on bars.

Donut charts:
- Thinner ring width (innerRadius ~70% of outerRadius) for minimal feel.
- No labels inside the donut. Center text for total/summary if needed.

Color strategy:
- Single-series charts: `--color-accent` (#CCFF00, chartreuse).
- Multi-series charts: Use category colors from the desaturated palette.
- Income vs expense: `--color-positive` for income, `--color-negative` for expense.

Tooltips:
- Card-style floating tooltip. Background: `--color-surface-elevated`. No border (elevation via background-shift). Border-radius: `--radius-md`. Font-size: 12px Body level.
- Amounts in tooltip use IBM Plex Mono (monospaced financial display).

**Inputs spec (replaces existing):**

Style: Underline-only for ALL inputs across all forms (not just modal forms). Bottom border only, no full box/rectangle.

Structure:
- Background: transparent. No `--color-surface` fill. The underline sits directly on whatever surface the form is on.
- Bottom border: 1px `--color-border-divider` (#222222) in default state.
- Text: `--color-text-primary` (#E8E8E8). Font-size 14px Body level.
- Padding: 10px 0 (vertical padding, no horizontal padding since no box).

Floating labels:
- Label starts as placeholder text on the underline in `--color-text-tertiary`.
- On focus or when filled: label floats up above the input, transitions to Label style (12px, uppercase, letter-spacing +2px, `--color-text-secondary`).
- Transition: 200ms ease for float animation.

Focus state:
- Underline transitions from `--color-border-divider` to `--color-accent` (#CCFF00, chartreuse).
- Transition: 200ms ease.
- Focus ring: standard 2px `--color-accent` outline per global focus ring spec.

Error state:
- Underline color: `--color-negative` (#FF3333).
- Error message below input in Meta level (11px, `--color-negative`).

Amount inputs:
- "$" prefix displayed in `--color-text-tertiary` at a slightly smaller size, positioned at the start of the underline. The prefix is static (does not float). Amount digits in IBM Plex Mono.

Percentage inputs:
- "%" suffix displayed in `--color-text-tertiary`, positioned at the end of the underline. Static suffix.

Select/dropdown inputs:
- Same underline style. Chevron icon (16px, `--color-text-tertiary`) at the right end of the underline.
