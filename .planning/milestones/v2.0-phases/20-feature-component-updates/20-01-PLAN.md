Create `src/components/ui/MoneyAmount.tsx` with this interface:

```typescript
interface MoneyAmountProps {
  /** Centavo string from API (e.g., "15075") */
  value: string
  /** Color direction: income (green), expense (red), neutral (default text) */
  variant?: 'income' | 'expense' | 'neutral'
  /** Additional CSS classes for the container span */
  className?: string
  /** Size variant for the "$" prefix text */
  size?: 'sm' | 'base' | 'lg' | 'xl' | '2xl'
}
```

Implementation:
- Call `formatMoney(value)` to get the formatted string (e.g., "$150.75")
- Split the result: extract the "$" character and the remaining digits
- Render two sibling `<span>` elements inside a wrapper `<span>`:
  - "$" span: `text-text-tertiary` color, one step smaller font size than the main amount (if size is "2xl" then "$" is "xl", if "lg" then "$" is "base", etc.)
  - Amount span: inherits the variant color, full `font-mono tabular-nums`
- Variant color mapping: income -> text-positive, expense -> text-negative, neutral -> text-text-primary
- The wrapper span gets `font-mono tabular-nums` plus the variant color class plus any custom className
- Default export, props interface exported as named export

Note: `formatMoney` from `@/lib/utils` uses `Intl.NumberFormat('es-MX', { style: 'currency', currency: 'MXN' })` which returns strings like "$150.75". The "$" is always the first character.
